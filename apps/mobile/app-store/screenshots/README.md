# App Store Screenshots

## Required Sizes

### iOS
- **iPhone 6.5"** (1284 x 2778px) - Required for App Store
- **iPhone 5.5"** (1242 x 2208px) - Required for older devices
- **iPad 12.9"** (2048 x 2732px) - Required if supporting iPad

### Android
- **Phone** (1080 x 1920px minimum, 16:9 recommended)
- **Feature Graphic** (1024 x 500px) - Required for Play Store listing

## Screenshot Descriptions

1. **Home Dashboard** - Shows the main tab with league standings, upcoming matchups, and quick actions
2. **Draft Room** - Active draft with player board, team needs, and pick timer
3. **Matchup & Scoring** - Live matchup view with real-time scoring updates
4. **Prediction Markets** - Market listing with odds, bet slip, and portfolio
5. **Player Comparison** - Side-by-side stat comparison with visual bars
6. **League Chat** - Chat interface with trash talk and trade discussion
7. **Keeper Selection** - Keeper draft interface with value ratings (Android only)
8. **Auction Draft** - Auction bidding interface with budget tracking (Android only)

## Design Guidelines

- Use dark theme (matches app default)
- Show realistic fantasy data (not placeholder)
- Include status bar with realistic time/battery
- Add caption text overlay describing key features
- Use consistent font and color scheme across all screenshots

## Generation

Screenshots can be generated using:
- Expo's screenshot tool
- Fastlane snapshot (iOS)
- Fastlane screengrab (Android)
- Manual capture from simulator/emulator
